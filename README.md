# Snipchat: Using screenshots to do stuff in computer

Just some experiments using Ollama and qwen3-vl:2b to do some funny things

### Screenchat
- Ask a question.
- Go to the window or screen that you want to ask questions about
- receive an answer from LLM.
Just testing whether LLM can be automated to read the screen and do work.

### Automator
- These scripts automate some tasks inside video games.
- when `automator.py` is executed, it presses the 1 key whenever a dialog choice that needs input appears.
- when `automator2.py` is executed, it clicks the screen whenever a prompt to "Press to continue" appears.
- Conclusion: qwen3 is kinda too slow to react, and it tends to overthink the response.
